# OFFSET

## 语法

**OFFSET**(引用,行偏移量,列偏移量[,行数,列数])  
**OFFSET**(reference,rows,cols,height,width)

### 参数

1. 引用 reference：引用的区域。
2. 行偏移量 rows：行偏移量。
    - 负数表示向上偏移。
    - 正数表示向下偏移。
3. 列偏移量 cols：列偏移量。
    - 负数表示向左偏移。
    - 正数表示向右偏移。
4. 行数 height：返回区域的行数。
    - 默认为传入引用的行数。
5. 列数 width：返回区域的列数。
    - 默认为传入引用的列数。

## 概述

引用区域偏移后的区域。

#### 偏移

```excel
OFFSET(A1,1,1)
```

返回：  
"张辽"

---

```excel
OFFSET(B2:F2,1,)
```

| 返回 |      |     |     |     |      |
