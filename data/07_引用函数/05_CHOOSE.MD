# CHOOSE

## 语法

**CHOOSE**(索引,值\_1[,值_2..值_254])  
**CHOOSE**(index,value1[,value2...value254])

### 参数

1. 索引 index：索引。
2. 值 value：索引对应的值。

## 概述

返回索引对应的值。

## 示例

### 示例数据

|     | A   | B   | C   |
| --- | --- | --- | --- |
| 1   | 1   | 2   | 3   |
| 2   | 4   | 5   | 6   |
| 3   | 7   | 8   | 9   |

```excel
CHOOSE(1,A1,B1:B3,{1,2,3},A1:C3)
```

返回：  
1

---

```excel
CHOOSE(2,A1,B1:B3,{1,2,3},A1:C3)
```

| 返回 |     |
| ---- | --- |
|      | 2   |
|      | 5   |
|      | 8   |

---

```excel
CHOOSE(3,A1,B1:B3,{1,2,3},A1:C3)
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 1   | 2   | 3   |
