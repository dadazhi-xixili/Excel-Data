# IF

## 语法

**IF**(逻辑表达式,真值,假值)  
**IF**(logical_test,value_if_true,[value_if_false])

### 参数

1. 逻辑表达式 logical_test：返回逻辑值或数值的表达式。
2. 真值 value_if_true：逻辑表达式为 TRUE 或 非零数值 时返回的值。
3. 假值 value_if_false：逻辑表达式为 FALSE 或 0 时返回的值。

## 概述

判断逻辑表达式的值，返回真值或假值。

## 补充

1. 在不输入逻辑表达式的情况下默认返回假值。
2. 真值默认为 0。
3. 假值省略为 FALSE，默认为 0。

## 示例

```excel
IF(1>2,"真","假")
过程：
IF(FALSE,"真","假")
输出：
假

IF(1-1,"真","假")
过程：
IF(0,"真","假")
输出：
假


IF(,)
输出：
FALSAE

IF(,,)
输出：
0
```
