# ISODD

## 语法

**ISODD**(值)  
**ISODD**(value)

### 参数

1. 值 value ：检验的值

## 概述

判断传入值是否为奇数。

## 示例

```excel
ISODD(1)
```

返回：  
TRUE

---

```excel
ISODD(2)
```

返回：  
FALSE

---

```excel
ISODD(1.2)
```

返回：  
TRUE

---

```excel
ISODD("")
```

返回：  
FALSE

---

```excel
ISODD(FALSE)
```

返回：  
FALSE
