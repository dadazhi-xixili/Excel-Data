# 文本

## 概述

是字节码组成的一总特殊数组.

### 补充

1. 字符串以双引号 `"` 包裹。
2. 字符串可以包含任意字符，包括空格、符号、汉字等。
3. 如果要在字符串中包含双引号，用两个双引号 `""` 来表示一个双引号。
   - `"hello ""world"""` = `hello "world"`
   - `""""""` = `""`
4. 字符有对应的编码，字符与编码之间可以通过函数转换。
   - `CHAR(65)` = `"A"`
   - `CODE("A")` = `65`
   - `CHAR(90)` = `"Z"`
   - `CODE("Z")` = `90`
   - `CHAR(97)` = `"a"`
   - `CODE("a")` = `97`
   - `CHAR(122)` = `"z"`
   - `CODE("z")` = `122`
   - `CHAR(48)` = `"0"`
   - `CODE("0")` = `48`
   - `CHAR(57)` = `"9"`
   - `CODE("9")` = `57`
