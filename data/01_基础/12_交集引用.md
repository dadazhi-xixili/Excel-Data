# 交集引用

## 表达式

**单元格 (空格)单元格**  
**RANGE (空格)RANGE**

**@单元格**  
**@RANGE**

## 概述

引用区域间交集区域。

### 补充

1. ` `(空格) 交集运算符
    - 引用区域引用间交集区域。
    - 无重叠区域将返回#NULL!。
2. @ 隐式交集符
    - 引用单行或单列区域与所在行或列号相同区域。
    - 对于数组取左上角第一个元素,但返回的是值而不是引用对象。数组向左侧和下侧溢出，与所在行列相同的区域只有左上角。
    - 无重叠区域将返回#VALUE!。

## 示例

### 示例数据

|     | A   | B   | C   | D   |
| --- | --- | --- | --- | --- |
| 1   | 1   | 2   | 3   |     |
| 2   | 4   | 5   | 6   |     |
| 3   | 7   | 8   | 9   |     |
| 4   |     |     |     |     |

### 交集运算

```excel
A1:C3 B2:D4
```

| 返回 |     |     |
| ---- | --- | --- |
|      | 5   | 6   |
|      | 8   | 9   |

### 隐式交集

```excel
D2=@A1:C3
```

D2 返回：  
4

```excel
D2=@A:A
```

D2 返回：  
4

```excel
B4=@2:2
```

B4 返回：  
5
