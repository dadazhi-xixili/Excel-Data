# 绝对引用

## 概述

绝对引用的公式引用的行号和列号不会随公式被复制到其他单元格而改变。

## 示例数据

|     | A   | B   | C   | D   |
| --- | --- | --- | --- | --- |
| 1   | 1   | 2   | 3   | 4   |
| 2   | 5   | 6   | 7   | 8   |
| 3   | 9   | 10  | 11  | 12  |
| 4   | 13  | 14  | 15  | 16  |

## 示例

|     | A    | B    | C    | D    |
| --- | ---- | ---- | ---- | ---- |
| 1   | =$A1 | =$A1 | =$A1 | =$A1 |
| 2   | =$A2 |      | →    | →    |
| 3   | =$A3 | ↓    |
| 4   | =$A4 | ↓    |

|     | A    | B    | C    | D    |
| --- | ---- | ---- | ---- | ---- |
| 1   | =A$1 | =B$1 | =C$1 | =D$1 |
| 2   | =A$1 |      | →    | →    |
| 3   | =A$1 | ↓    |
| 4   | =A$1 | ↓    |

|     | A      | B      | C      | D      |
| --- | ------ | ------ | ------ | ------ |
| 1   | =$A\$1 | =$A\$1 | =$A\$1 | =$A\$1 |
| 2   | =$A\$1 |        | →      | →      |
| 3   | =$A\$1 | ↓      |
| 4   | =$A\$1 | ↓      |

### 表示方式：

在列标和行号前加 \$，如 \$A\$1、\$B\$2。

### 特点：

无论公式复制到哪里，引用始终保持不变。
