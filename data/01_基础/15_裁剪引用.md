# 裁剪引用

## 表达式

**区域.:.区域**  
**RANGE.:.RANGE**

## 概述

引用裁剪连续空白区域后的区域。

## 示例

### 示例数据

|     | A   | B   | C   | D   | E   |
| --- | --- | --- | --- | --- | --- |
| 1   |     |     |     |     |     |
| 2   |     |     | 2   |     |     |
| 3   |     | 4   |     | 6   |     |
| 4   |     |     | 8   | 9   |     |
| 5   |     |     |     |     |     |

### 开始区域截取

```excel
A1.:E5
```

| 返回 |     |     |     |     |
| ---- | --- | --- | --- | --- |
|      | 0   | 2   | 0   | 0   |
|      | 4   | 0   | 6   | 0   |
|      | 0   | 8   | 9   | 0   |
|      | 0   | 0   | 0   | 0   |

### 结束区域截取

```excel
A1:.E5
```

| 返回 |     |     |     |     |
| ---- | --- | --- | --- | --- |
|      | 0   | 0   | 0   | 0   |
|      | 0   | 0   | 2   | 0   |
|      | 0   | 4   | 0   | 6   |
|      | 0   | 0   | 8   | 9   |

### 组合裁剪

```excel
A1.:.E5
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 0   | 2   | 0   |
|      | 4   | 0   | 6   |
|      | 0   | 8   | 9   |
