# AVEDEV

## 语法

**AVEDEV**(值\_1[,值_2...值_255])  
**AVEDEV**(number1[,number2...number255])

### 参数

1. 值 numner ：要求平均值计算的值。

## 概述

计算平均偏差值。

## 示例

|     | A     | B     |
| --- | ----- | ----- |
| 1   | =TRUE | ="10" |

```excel
AVEDEV(TRUE,2)
AVEDEV("1",2)
```

输出： 0.5

```excel
AVEDEV(A1,2,4)
```

输出： 1
