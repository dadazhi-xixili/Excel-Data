# MODE.MULT

## 语法

**MODE.MULT**(数值 1[,数值 2...数值 255])  
**MODE.MULT**(number1[,number2...number255])

### 参数

1. 数值 number ：要计算众数的数值或数组。

## 概述

返回一组数据中出现频率最高的数值的垂直数组。如果有多个众数，将返回所有众数。

### 补充

1. 如果数据集中没有重复的数值，将返回 #N/A 。
2. 文本、逻辑值和空单元格将被忽略。

## 示例

## 示例数据

|     | A   | B   | C   | D   |
| --- | --- | --- | --- | --- |
| 1   | 1   | 2   | 3   | 4   |
| 2   | 2   | 3   | 3   | 5   |
| 3   | 3   | 4   | 5   | 5   |

```excel
MODE.MULT(A1:D3)
```

返回：  
3

---

```excel
MODE.MULT({1,1,2,2,3,4,5},"3")
```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 2   |
