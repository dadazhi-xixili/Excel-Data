# COUNTA

## 语法

**COUNTA**(值\_1[,值_2...值_255])  
**COUNTA**(value_1[,value_2...value_255])

### 参数

1. 值 value：要统计的数组或单元格。

## 概述

统计非空值。

## 示例

|     | A     | B     |
| --- | ----- | ----- |
| 1   | ="A"  | =1    |
| 2   | =NA() | =TRUE |
| 3   | =""   |       |

```excel
COUNTA(A1:B3)
```

输出：  
5
