# AVERAGE

## 语法

**AVERAGE**(值 1[,值 2...值 255])  
**AVERAGE**(number1[,number2...number255])

### 参数

1. 值 number ：要平均值的值。

## 概述

对传入的数值求平均值。

### 补充

1. 直接传入的文本会尝试转换为数值后求平均值，无法转换值将被忽略。
2. 数组或单元格的文本不会尝试转换。
3. 以数值格式写入单元格后再转换为文本的数值仍可被识别为数值。

## 示例数据

|     | A    | B   |
| --- | ---- | --- |
| 1   | ="4" | =40 |

```excel
AVERAGE(A1,B1,400,"4000",{"40000",400000})
```

输出：  
101110

```excel
AVERAGE(TRIE)
```

输出：  
1
