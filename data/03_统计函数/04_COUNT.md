# COUNT

## 语法

**COUNT**(值\_1[,值_2...值_255])  
**COUNT**(value1[,value2...value255])

### 参数

1. 值 value ：计数的值。

## 概述

对数值计数。

### 补充

1. 直接传入的文本会尝试转换为数值后计数。
2. 数组或单元格的文本不会尝试转换。
3. 以数值格式写入单元格后再转换为文本的数值仍可被识别为数值。

## 示例

## 示例数据

|     | A    | B   |
| --- | ---- | --- |
| 1   | ="1" | =20 |

```excel
COUNT(A1)
```

输出：  
0

```excel
COUNT(A1,B1)
```

输出：  
1

```excel
COUNT(A1,B1,300,"4000")
```

输出：  
3

```excel
COUNT(A1,B1,300,"4000",{"50000",600000})
```

输出：  
4
