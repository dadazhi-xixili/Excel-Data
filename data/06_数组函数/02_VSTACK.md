# VSTACK

## 语法

**VSTACK**(数组\_1[,数组_2...])  
**VSTACK**(array_1[,array_2...])

### 参数

1. 数组 array：要拼接的数组。

## 概述

纵向拼接数组。

## 补充

1. 大小不一致的数组空缺部分由#N/A 填充。

## 示例

```excel
VSTACK(1,2,3,4,5)
```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 2   |
|      | 3   |
|      | 4   |
|      | 5   |

---

```excel
VSTACK({1,2,3},{4,5,6})
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 1   | 2   | 3   |
|      | 4   | 5   | 6   |

---

```excel
HSTACK({1,2},3)
```

| 返回 |     |      |
| ---- | --- | ---- |
|      | 1   | 2    |
|      | 3   | #N/A |
