# SEQUENCE

## 语法

**SEQUENCE**(行数[,列数=1,初始值=1,步进=1])  
**SEQUENCE**(rows[,columns=1,start=1,step=1])

### 参数

1. 行数 rows：生成的行数。
2. 列数 columns：生成的列数。
3. 初始值 start：生成的数组的起始值。
4. 步进 step：生成的数组的步进值。

## 概述

从初始值生成序列数组。

## 示例

### 示例数据

```excel
SEQUENCE(3)
```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 2   |
|      | 3   |

---

```excel
SEQUENCE(3,3)
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 1   | 2   | 3   |
|      | 4   | 5   | 6   |
|      | 7   | 8   | 9   |

---

```excel
SEQUENCE(3,3,10)
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 10  | 11  | 12  |
|      | 13  | 14  | 15  |
|      | 16  | 17  | 18  |

---

```excel
SEQUENCE(3,3,10,10)
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 10  | 20  | 30  |
|      | 40  | 50  | 60  |
|      | 70  | 80  | 90  |
