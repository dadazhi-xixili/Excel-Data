# TRANSPOSE

## 语法

**TRANSPOSE**(数组)  
**TRANSPOSE**(array)

## 概述

转置数组。

### 参数

1. 数组 array：要转置的数组。

## 示例

### 示例数据

|     | A   | B   | C   |
| --- | --- | --- | --- |
| 1   | 1   | 2   | 3   |
| 2   | 4   | 5   | 6   |
| 3   | 7   | 8   | 9   |

```
TRANSPOSE(A1:C3)
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 1   | 4   | 7   |
|      | 2   | 5   | 8   |
|      | 3   | 6   | 9   |

---

```
TRANSPOSE(TRANSPOSE((A1:C3)))
```

| 返回 |     |     |     |
| ---- | --- | --- | --- |
|      | 1   | 2   | 3   |
|      | 4   | 5   | 6   |
|      | 7   | 8   | 9   |
