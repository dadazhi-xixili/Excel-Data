# WRAPCOLS

## 语法

**WRAPCOLS**(数组,基准数量[,空缺填充=#N/A])  
**WRAPCOLS**(vector,wrap_count[,pad_with=#N/A])

### 参数

1. 数组 vector：转换的一维数组。
2. 基准数量 wrap_count：列内元素数量。
3. 填充值 pad_with：填充值。

## 概述

将一维数组折叠为二维数组，以列方式按指定元素数量限制折叠。

## 示例

### 示例数据

|     | A   | B   | C   |
| --- | --- | --- | --- |
| 1   | 1   | 2   | 3   |

```excel
WRAPCOLS(A1:C1,2)
```

| 返回 |     |      |
| ---- | --- | ---- |
|      | 1   | 3    |
|      | 2   | #N/A |

```excel
WRAPCOLS(A1:C1,2,"空缺")
```

| 返回 |     |      |
| ---- | --- | ---- |
|      | 1   | 3    |
|      | 2   | 空缺 |
