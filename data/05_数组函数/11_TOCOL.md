# TOCOL

## 语法

**TOCOL**(数组[,模式=0,方向=FALSE])  
**TOCOL**(array[,ignore=0,scan_by_column=FALSE])

### 参数

1. 数组 array：转换的数组。
2. 模式 ignore：行为模式。
   - 0 保留所有值。
   - 1 忽略空白。
   - 2 忽略错误。
   - 3 忽略空白和错误。
3. 方向 scan_by_column：扫描方向。
   - TRUE 按列扫描。
   - FALSE 按行扫描。

## 概述

数组转换为一维数组，单列的方式。

## 示例

### 示例数据 1

|     | A   | B   | C   |
| --- | --- | --- | --- |
| 1   | 1   | 2   | 3   |
| 2   | 4   | 5   | 6   |
| 3   | 7   | 8   | 9   |

```excel
TOCOL(A1:C3)
```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 2   |
|      | 3   |
|      | 4   |
|      | 5   |
|      | 6   |
|      | 7   |
|      | 8   |
|      | 9   |

```excel
TOCOL(A1:C3,,1)
```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 4   |
|      | 7   |
|      | 2   |
|      | 5   |
|      | 8   |
|      | 3   |
|      | 6   |
|      | 9   |

### 示例数据 2

| --- | ------ | ------- | ---- |
| 1   | 1      |         | #N/A |

| 返回 |         |

|      | 1       |
|      | #N/A    |
|      | #CALC!  |
|      | #DIV/0! |
|      | 6       |

```excel

```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 0   |
|      | 6   |
|      | 0   |
|      | 0   |
|      | 0   |

```excel
TOCOL(A1:C3,3)
```

| 返回 |     |
| ---- | --- |
|      | 1   |
|      | 6   |
